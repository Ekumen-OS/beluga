Detailed Description

template<typename Derived>
class beluga::BaseRegularGrid2< Derived >

Regularly spaced 2D grid base type.

When instantiated, it satisfies Beluga named requirements: RegularGrid2.

Template Parameters
DerivedConcrete regular grid type. It must define Derived::resolution(), as described in Beluga named requirements: RegularGrid2.
