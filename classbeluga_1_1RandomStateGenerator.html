template<class Mixin>
class beluga::RandomStateGenerator< Mixin >

A random state generator.

An implementation of Beluga named requirements: StateGenerator.

Template Parameters
MixinThe mixed-in type. An instance m of Mixin must provide:
  • A make_random_state() method that satisfies the requirements specified in Beluga named requirements: SensorModel.
