Detailed Description

template<class Mixin, class Container, class Particle = typename Container::value_type, class State = typename particle_traits<Particle>::state_type, class Weight = typename particle_traits<Particle>::weight_type>
class beluga::StoragePolicy< Mixin, Container, Particle, State, Weight >

Configurable storage policy.

This class implements StorageInterface and satisfies Beluga named requirements: StoragePolicy.

Template Parameters
MixinThe mixed-in type. An instance m of Mixin must provide:
  • A max_samples() method that satisfies the requirements specified in Beluga named requirements: Limiter.
ContainerThe particle container type. It must satisfy Beluga named requirements: ParticleContainer.
ParticleThe particle type, deduced from the container value type.
StateThe state type of a particle.
WeightThe weigth type of a particle.
