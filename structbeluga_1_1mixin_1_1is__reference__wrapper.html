Detailed Description

template<typename T>
struct beluga::mixin::is_reference_wrapper< T >

Type trait to detect that a given type is a reference wrapper.

If T (after decay) is an instantiation of std::reference_wrapper, provides a member constant value equal to true. For any other type, value is false.
