Detailed Description

template<typename T>
struct beluga::mixin::is_descriptor< T >

Type trait to detect that a given type is a descriptor.

If T (after decay) is an instantiation of beluga::mixin::descriptor, provides a member constant value equal to true. For any other type, value is false.
