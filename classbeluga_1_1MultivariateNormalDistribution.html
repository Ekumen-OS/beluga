Detailed Description

template<class Matrix>
class beluga::MultivariateNormalDistribution< Matrix >

Multivariate normal distribution.

MultivariateNormalDistribution<Matrix> is an implementation of the RandomStateDistribution named requirements that represents a multi-dimensional normal distribution of real-valued random variables.

Template Parameters
MatrixThe type of the covariance matrix; this is expected to be an instantiation of the Eigen::Matrix class template.
